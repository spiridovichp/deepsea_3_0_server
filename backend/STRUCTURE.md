# Структура Backend приложения

## Описание директорий

### Корневая директория (`backend/`)

```
backend/
├── src/                    # Исходный код приложения
├── tests/                  # Тесты
├── docs/                   # Документация
├── scripts/                # Вспомогательные скрипты
└── logs/                   # Логи приложения
```

### Исходный код (`src/`)

#### `src/api/` - API слой приложения

- **`controllers/`** - Контроллеры (обработчики HTTP запросов)
- **`routes/`** - Определение маршрутов API
- **`services/`** - Бизнес-логика приложения
- **`middleware/`** - Промежуточное ПО (аутентификация, валидация, обработка ошибок)
- **`validators/`** - Валидаторы запросов
- **`modules/`** - Модули по функциональным областям:
  - `auth/` - Аутентификация и авторизация
  - `users/` - Управление пользователями
  - `roles/` - Управление ролями и разрешениями
  - `projects/` - Управление проектами
  - `stages/` - Управление этапами проекта
  - `issues/` - Управление задачами
  - `documents/` - Управление документами
  - `customer-questions/` - Вопросы от заказчика
  - `materials/` - Управление материалами
  - `equipment/` - Управление оборудованием
  - `specifications/` - Управление спецификациями
  - `statements/` - Управление ведомостями
  - `storage/` - Управление файлами
  - `messages/` - Сообщения и комментарии
  - `notifications/` - Уведомления
  - `wiki/` - Wiki система
  - `specializations/` - Справочник специализаций

#### `src/config/` - Конфигурация приложения

- Конфигурационные файлы для разных окружений
- Настройки базы данных
- Настройки внешних сервисов

#### `src/db/` - Работа с базой данных

- **`migrations/`** - Миграции базы данных
- **`seeds/`** - Начальные данные (seeders)
- **`models/`** - Модели данных (ORM)

#### `src/utils/` - Утилиты

- Вспомогательные функции
- Хелперы
- Утилиты для работы с датами, строками и т.д.

#### `src/types/` - Типы TypeScript

- Определения типов и интерфейсов
- Типы для запросов и ответов

#### `src/errors/` - Обработка ошибок

- Кастомные классы ошибок
- Обработчики ошибок

### Тесты (`tests/`)

- **`unit/`** - Юнит-тесты
- **`integration/`** - Интеграционные тесты
- **`e2e/`** - End-to-end тесты

### Документация (`docs/`)

- API документация
- Документация по архитектуре
- Руководства для разработчиков

### Скрипты (`scripts/`)

- Скрипты для миграций
- Скрипты для деплоя
- Вспомогательные скрипты

### Логи (`logs/`)

- Файлы логов приложения
- Логи ошибок
